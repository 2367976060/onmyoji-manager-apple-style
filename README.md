# 阴阳师账号管理器

一款采用 **Apple 设计风格** 的阴阳师游戏账号管理工具，支持账号管理、式神信息记录和数据统计。

## ✨ 特性

### 🎨 Apple 风格设计
- **磨砂玻璃效果 (Glassmorphism)** - 半透明毛玻璃质感
- **圆润圆角设计** - 符合 iOS/macOS 的视觉语言
- **清新配色方案** - 以白色、浅灰色为主，蓝色点缀
- **流畅过渡动画** - 丝滑的交互动画效果
- **响应式布局** - 完美适配各种屏幕尺寸

### 📱 核心功能
- **账号管理** - 添加、编辑、删除游戏账号
- **角色信息** - 记录等级、经验、金币、勾玉等数据
- **式神管理** - 式神列表、稀有度、等级、技能、御魂配置
- **数据统计** - 实时统计展示
- **本地存储** - 数据保存在浏览器本地，无需服务器
- **搜索功能** - 快速搜索账号和式神

## 🎯 使用说明

### 账号管理
1. 点击「添加账号」按钮
2. 填写账号名称、等级、金币、勾玉等信息
3. 支持编辑和删除操作
4. 所有数据自动保存到本地

### 式神管理
1. 切换到「式神管理」视图
2. 选择要管理的账号
3. 点击「添加式神」
4. 填写式神名称、稀有度、等级、技能、御魂配置

### 数据统计
- 总账号数
- 平均等级
- 总勾玉数
- 总金币数

## 💾 数据存储

所有数据保存在浏览器的 LocalStorage 中：
- 账号数据：`onmyoji_accounts`
- 式神数据：`onmyoji_shikigami`

**注意**：清除浏览器数据会丢失所有记录，请定期备份。

## 🎨 设计特点

### Glassmorphism 效果
```css
background: rgba(255, 255, 255, 0.72);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.5);
```

### 配色系统
- **主色调**：`#007aff` (Apple Blue)
- **背景**：渐变浅灰色
- **文字**：深灰色、浅灰色层次
- **强调色**：红色、橙色、绿色用于状态标识

### 圆角规范
- 小型组件：8px
- 中型组件：12px
- 大型卡片：16px
- 弹窗容器：20px
